title: 《廖雪峰Python教程》电子书制作
date: 2017-2-6 14:01:55
categories: Python
tags: 

- 爬虫
- kindle

廖雪峰的Python教程是很不错的Python入门资料，可惜只有网页形式，只能在线阅读，如果能将其转化为mobi电子书放在kindle上阅读，则会方便很多。本文记录了如何使用Python及其他工具抓取廖雪峰的Python教程制作电子书的过程,也正好学以致用。

工具

编程语言 Python

Python版本：3.5
第三方库：

  • requests 代替Python官方的 urllib.request 请求内容
  • BeautifulSoup 解析网页信息，定位目标内容

编辑器 Sublime Text 3

Sublime Text 3作为文本编辑器，此外也用到了包 OmniMarkupPreviewer，用于给抓取的内容添加简单的CSS格式。

Calibre

Calibre是一个免费的电子书管理工具，此处用于将网页文件转化成电子书格式。

抓取内容

分析网页

抓取内容首先要明确要抓取的内容在哪里，因为不是所有内容都集中在一个网页上，而且一个网页上的内容也不会全是你想要的，所以首先要做的事情有两件：

  1. 得到要抓取内容所在网页的URL集合（如果有专门的索引页就比较省事）
  2. 在网页中找到抓取内容所在的div容器

在chrome中打开Python教程，可以看到页面左侧有目录边栏，右边主体部分为教程正文内容。

通过Ctrl+U组合键查看网页源代码,可以找到以下信息：

目录信息包含在类名class为 “uk-nav uk-nav-side” style属性为 “margin-right:-15px;” 的无序列表ul中。此外目录有以下特点：

  • 目录有级别大小，如“Python教程”为最高级，“Python简介”为一级目录，“Python解释器”是“安装Python”的二级目录。
  • 目录的级别通过所在元素的右边外边距位移量 (margin-right)设置来区分，如最高级目录无位移量，一级目录为 “1em”， 二级目录为 “2em”，三级目录为 “3em”。


正文内容包含在类名class为 “x-wiki-content” 的div容器中。可以发现，正文中有时会以<img>的形式包含一些图片链接（部分教程内容甚至还有视频），如果要制作电子书，这些图片文件也必须同时下载下来。

目录必要信息包括标题，URL和级别。标题可以直接获取，URL则需要增加域名前缀，而目录级别此处是通过正则表达式提取li元素的style属性值中的数值来设定（最高级目录“Python教程”则设定为默认的1级目录）。

在代码中 requests 的使用只是简单地向URL请求内容并返回网页源码交给 BeautifulSoup 解析，其实给 requests.get(url) 方法增加请求头 header 可以更真实地模拟浏览器的请求方式，此处省略。

代码中更多的是涉及 BeautifulSoup 的使用,具体可参考其官方文档，简单介绍如下：

  • soup.find_all(name, attrs) 搜索当前tag的所有子节点，选择符合过滤器条件的所在内容并以列表的形式返回
  • sop.find(name.attrs) 搜索当前tag的所有子节点，选择第一个符合过滤器条件的内容并返回
  • tag.has_attr(attr_name) 判断当前tag是否包含 attr_name 属性，并返回相应布尔值
  • tag.subtag 返回当前节点的 subtag 子节点
  • tag[‘attr’] 返回当前节点的名为’attr’的属性内容
  • tag.prettify 返回当前节点的格式化HTML文本

制作电子书

生成HTML文件

在安装了 OmniMarkupPreviewer 包的 Sublime Text 3中打开生成的md文件，右键选择 “Export Markup as HTML” 或直接快捷键 Ctrl+Alt+X ，在当前目录生成HTML文件。

为什么不直接生成HTML文件，而要如此大费周章转化呢？个人觉得，如果直接生成HTML文件，还需额外添加CSS样式，否则在下一步Calibre制作的电子书在格式上会比较难看（当然，Calibre可能会有自定义的CSS样式添加，此处未深入研究），而Markdown本身兼容HTML语法，且 OmniMarkupPreviewer 支持将md文件生成比较清爽风格的HTML文件预览，所以就直接采用了。

一句话，其实就是最近用ST写markdown比较熟，就顺便如此折腾:-D

Calibre 生成电子书

将前文生成的HTML文件与img文件夹打包添加到Calibre的书库中，右键选择“转化书籍”，输入格式为 ZIP， 输出格式为 MOBI，并添加一些自定义内容，如书名，作者，封面图像等等。

选择“内容目录”，添加目录，此处就用到了类名的“toc”的标题元素。

点击“确定”，生成mobi格式电子书，以下为在Kindle上的使用截图。
